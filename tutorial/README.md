# Rust 区块链开发完全教程

> 从零开始用 Rust 构建一个完整的区块链系统

## 课程大纲

### 第一阶段：基础篇

| 章节 | 内容 | 目标 |
|------|------|------|
| [01-区块结构](01-block-structure/) | 定义区块数据结构 | 理解区块的基本组成 |
| [02-哈希计算](02-hash/) | 实现 SHA256 哈希 | 掌握加密哈希算法 |

| [04-工作量证明](04-pow/) | 实现 POW 挖矿 | 理解共识机制 |

### 第二阶段：密码学与账户

| 章节 | 内容 | 目标 |
|------|------|------|

### 第三阶段：交易模型

| 章节 | 内容 | 目标 |

| 章节 | 内容 | 目标 |
|------|------|------|
| [09-API接口](09-api/) | 区块浏览器 API | 学习 Web 开发 |

## 学习方法

1. **按顺序学习**：每章建立在前一章基础上
2. **动手实践**：每章都有完整的可运行代码
3. **理解原理**：不只是写代码，更要理解为什么
4. **逐步完善**：从简单开始，逐步添加功能
