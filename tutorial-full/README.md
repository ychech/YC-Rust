## 环境准备

```bash
# 安装 Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# 安装依赖
sudo apt-get update
sudo apt-get install -y build-essential pkg-config libssl-dev
```

## 快速开始

```bash
# 进入任意章节
cd tutorial-full/01-basic
cargo run
```
