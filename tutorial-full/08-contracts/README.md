# 08. 智能合约 - Token、NFT、DEX

## 本章目标

实现三种核心智能合约：代币、NFT、去中心化交易所。

## 合约类型

### 1. Token合约（YRC-20）
```rust
/// 同质化代币标准
/// 
/// 功能：
/// - 查询余额
/// - 转账
/// - 授权
/// - 查询总供应量
```

### 2. NFT合约（YRC-721）
```rust
/// 非同质化代币标准
/// 
/// 功能：
/// - 铸造唯一代币
/// - 转移所有权
/// - 查询所有者
/// - 授权管理
```

### 3. DEX合约（AMM）
```rust
/// 自动做市商
/// 
/// 功能：
/// - 添加流动性
/// - 代币交换
/// - 恒定乘积公式: x * y = k
```
