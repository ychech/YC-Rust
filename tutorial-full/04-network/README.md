## 什么是P2P网络？

P2P（Peer-to-Peer）网络是一种去中心化的网络架构，所有节点地位平等。

```
传统客户端-服务器:
    客户端1 ──→ 服务器 ←── 客户端2
    客户端3 ──→       ←── 客户端4

P2P网络:
    节点1 ←────→ 节点2
      ↑ ↘     ↗ ↑
       ←─ 节点3 ─→
```
